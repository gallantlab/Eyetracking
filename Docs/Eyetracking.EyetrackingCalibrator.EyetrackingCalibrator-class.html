FindPupils(self, window=None, blur=5, dp=1, minDistance=600, param1=80, param2=20, minRadius=15, maxRadius=22, windowSize=15, outlierThresholds=None, filterPupilSize=True, surfaceBlur=None, nThreads=None)

source code 

Finds pupil traces

Parameters:
  • window - 4-ple<int>?, subwindow in frame to examine, order [left, right, top, bottom]
  • blur - int, median blur filter width
  • dp - float, inverse ratio of accumulator resolution to image resolution
  • minDistance - float, min distance between centers of detected circles
  • param1 - float, higher threshold for canny edge detector
  • param2 - float, accumulator threshold at detection stage, smaller => more errors
  • minRadius - int, min circle radius
  • maxRadius - int, max circle radius
  • windowSize - int, median filter time window size
  • outlierThresholds - list<float>?, thresholds in percentiles at which to nan outliers, if none, does not nan outliers
  • filterPupilSize - bool, filter pupil size alone with position?
  • surfaceBlur - int?, if present, radius to use for surface blur
  • nThreads - int?, number of threads to use for pupil finding. if none, use all cores

source code 

Estimates the pupil positions corresponding to each calibration point

Parameters:
  • beginTime - 4ple<int>?, time of calibration sequence onset TODO: replace this with a frame number
  • method - function, method used to aggregated points to one summary point
  • startDelay - float, time delay in seconds to account for eye movement delay
  • endSkip - float, time in seconds to clip from the end of a fixation period
  • duration - float|list<float>?, duration of each fixation point, if list, search through all and select best
  • filtered - bool, use filtered pupil locations?
  • verbose - bool, print stuff?
