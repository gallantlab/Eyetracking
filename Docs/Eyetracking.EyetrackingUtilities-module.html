ParseHistoryForStartTTLs(historyFileName, useMovieMarkers=True, TR=2.0, onset=False, threshold=1.5)

source code 

Parses the history file from Avotec for the start TTL timings for runs

Parameters:
  • historyFileName (str) - name of history file from avotec
  • useMovieMarkers (bool) - use the start/stop save movie entries to calculate runs? if true, the TR, onset, and threshold arguments are useless
  • TR (float) - TR length used
  • onset (bool) - use the TTL pulse HI instead of the LO value?
  • threshold (float) - multiple of the TR interval to use as a threshold as a break?
Returns: list<tuple<tuple<float>, int>>
first value is the timestamp of the first TTL in a run, and the second is number of TRs in each run
